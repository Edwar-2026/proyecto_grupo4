Este repositorio contiene el desarrollo del proyecto final del curso, estructurado en módulos para el procesamiento, análisis y modelado de datos.

## 📁 Estructura del Repositorio
* **`data/`**: Contiene los conjuntos de datos oficiales en formato CSV utilizados para el análisis.
* **`notebooks/`**: Cuadernos de trabajo de Google Colab (Partes 1 a 5) con el análisis exploratorio, limpieza y modelado.
* **`src/`**: Scripts de Python con el código fuente optimizado y el pipeline de ejecución.

## 🛠️ Tecnologías Utilizadas
* Python (Pandas, NumPy, Scikit-learn)
* Google Colab / Jupyter Notebooks
* Git & GitHub
